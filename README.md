# 解析函数的共形变换可视化（Conformal-Mapping-Visualization）

用 Python 可视化复变函数中的**共形变换**or**保角变换**

> 课程：数据结构与算法B ｜ 作者：陈志胤｜ 专业：物理学院

## 一、项目背景

这学期我在学《数学物理方法》的复变函数部分.这部分有一个很漂亮的结论：
解析函数在导数非零处是**共形映射**——它会把相交的曲线变换成另外两条曲线，且**保持交角不变**.
由于只在脑海中想象会比较抽象，故我想将它可视化：
让网格、圆或是一张真实照片经过 `z²`、`eᶻ`等函数后变形，直观感受"解析函数如何重新塑造平面".

## 二、核心算法与知识点

| 用到的知识点        | 在项目里怎么体现                                    |
| ------------------- | --------------------------------------------------- |
| 复数的表示与运算    | 全程用 `numpy` 复数数组直接做 `f(z)` 计算           |
| 解析函数 / 共形映射 | 变换前后网格交点处仍互相垂直，直观验证保角性        |
| 逆映射与逆向采样    | 照片变换：对每个输出点 `w` 取原图 `f⁻¹(w)` 处的颜色 |

- **网格变换**（`conformal_grid.py`）：把横竖直线 / 同心圆密集采样成点，逐点算
  `f(z)`，再画出来.
- **照片变换**（`conformal_image.py`）：采用**逆向采样**.
  输出画面上每个像素 `w`，其颜色来自原图的 `f⁻¹(w)` 处，用双线性插值取色.
